## Featured Robotics Projects

### 1) Robot Visual Tracking and Communication System *(Master’s Thesis — In Progress)*
- Designed a multi-microcontroller architecture: **Raspberry Pi → ESP32 → STM32** for real-time perception-to-control data flow.
- Defined latency requirements from control constraints with a target of **< 30 ms**.
- Implemented and profiled **SPI, UART, and ESP-NOW** to optimize end-to-end latency (**currently ~50 ms**).
- Identified bottlenecks in the image capture/processing pipeline and developed optimization strategies.
- Built Python-based real-time visualization tools for performance monitoring and debugging.

### 2) Autonomous Mobile Robot
- Developed a real-time embedded control system using **MicroPython + STM32** with IMU, encoders, and IR sensors.
- Designed an observer-based estimator and model-based controller for stable autonomous navigation.
- Optimized the control loop to consistently meet a **25 ms** timing constraint.
- Reduced course completion time from **~30 s to ~15 s** (fastest of 12 teams) through iterative profiling and control tuning.
- Built Bluetooth telemetry for live pose streaming and rapid debugging plots.

### 3) 3-DOF Robotic Harvester System
- Designed and implemented embedded control for a 5-motor manipulator with trajectory planning and safety features.
- Debugged motor instability with oscilloscope-based PWM analysis and traced issues to software-timed PWM.
- Switched to **DMA-based PWM generation**, improving signal stability and controller performance.
- Performed kinematic modeling (DH parameters) and control tuning for reliable operation.
